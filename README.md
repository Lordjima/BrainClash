# BrainClash

BrainClash est une webapp de quiz multijoueur en temps réel avec :
- front React + Vite
- backend Express
- temps réel via Socket.IO
- stockage MySQL
- authentification Twitch

## Stack
- React 19
- Vite
- TypeScript
- Express
- Socket.IO
- MySQL2

## Installation

```bash
npm install
cp .env.example .env
